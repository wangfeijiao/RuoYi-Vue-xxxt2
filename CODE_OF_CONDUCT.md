# Code of Conduct

## Our Pledge

为维护一个开放、专业、可协作的社区环境，参与本项目的维护者与贡献者承诺：

- 尊重不同背景、经验与观点
- 聚焦问题本身，而不是针对个人
- 以可执行、可验证的方式讨论技术方案
- 避免攻击性、羞辱性、骚扰性表达

## Our Standards

有助于建设社区的行为包括：

- 提供清晰、具体、可复现的问题描述
- 在提出变更时解释动机、范围和影响
- 尊重项目中既有的角色边界、门禁规则和 artifact 规范
- 对评审意见进行理性回应，并用事实支撑判断

不可接受的行为包括：

- 人身攻击、侮辱、歧视、骚扰
- 恶意破坏讨论、反复制造无效争论
- 蓄意绕过项目协作规则并拒绝说明原因
- 泄露他人隐私、敏感信息或未授权内容

## Enforcement Responsibilities

项目维护者负责解释并执行本行为准则，并可对违反者采取适当措施，包括但不限于：

- 提醒与警告
- 拒绝或关闭相关 issue / pull request
- 暂停或取消社区参与资格

## Scope

本行为准则适用于：

- 项目仓库中的 issue、discussion、pull request、review
- 与本项目直接相关的公开技术交流场景

## Reporting

如遇到严重违规行为，请联系项目维护者，并提供必要上下文以便处理。
