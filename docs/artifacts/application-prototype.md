## 2. 页面原型说明

### 2.1 应用台账列表页

#### 页面目标

让应用管理员和管理层快速筛选、查询和进入应用详情。

#### 页面结构

- 顶部筛选区
- 中部列表区
- 右侧快捷操作或批量操作区

#### 筛选项

- 应用名称/编码
- 建设来源：自建、集团统建、交通部统建
- 应用类型
- 分级分类
- 运行状态
- 所属单位
- 责任人

#### 列表字段

- 应用编码
- 应用名称
- 建设来源
- 应用类型
- 分级分类
- 所属单位
- 主责任人
- 所属项目
- 当前状态
- 异常标记
- 最近更新时间

#### 顶部操作

- 新增应用
- 导入台账
- 导出结果
- 查看统计

#### 行内操作

- 查看详情
- 编辑
- 查看通知
- 查看依赖

### 2.2 应用新增/编辑页

#### 页面目标

完成应用基本台账与责任矩阵的录入。

#### 表单分区

- 基础信息区
- 建设来源与分类区
- 归属与项目区
- 责任矩阵区
- 技术与访问区
- 依赖摘要区

#### 基础字段

- 应用编码
- 应用名称
- 应用类型
- 应用状态
- 访问地址
- 技术栈

#### 分类字段

- 建设来源
- 分级分类
- 所属单位

#### 责任字段

- 业主方负责人
- 开发单位
- 开发单位负责人
- 统一认证团队负责人
- IT 资源团队负责人
- 数据资源团队负责人
- 网络安全团队负责人

#### 交互要求

- 保存前校验应用编码唯一性
- 项目字段提供搜索选择，不要求用户手填 ID
- 若尚未补齐关键责任人，提交时给予阻断提示

### 2.3 应用详情页

#### 基础信息页签

- 展示应用台账核心信息
- 支持查看所属项目、访问地址、技术栈、建设来源、当前状态

#### 责任矩阵页签

- 以清晰表格或卡片方式展示各责任角色
- 支持查看通知接收对象与升级关系

#### 依赖关系页签

- 以分组清单展示项目、IT 资源、数据库/API、网络资源
- 每项依赖对象显示名称、状态、责任人、跳转入口
- 预留影响范围图谱展示区域

#### 运行概览页签

- 顶部展示应用健康摘要卡片
- 中部展示底层 IT 资源状态与网络状态
- 下部展示最近异常事件与处理结果

#### 通知记录页签

- 展示通知时间、类型、接收人、内容摘要、处理状态
- 支持筛选“未处理/已处理/升级中”

#### 操作日志页签

- 记录台账修改、责任变更、依赖调整、状态更新等日志

### 2.4 应用统计分析页

#### 页面目标

满足管理层和应用管理员的治理视图。

#### 核心模块

- 应用总量与异常总量 KPI 卡片
- 建设来源分布
- 分级分类分布
- 所属单位分布
- 状态分布
- 责任团队风险分布

### 2.5 应用通知中心页

#### 页面目标

集中处理应用相关异常与通知。

#### 列表字段

- 通知类型
- 应用名称
- 影响对象
- 异常时间
- 接收人
- 当前处理状态

#### 操作

- 查看详情
- 标记处理中
- 标记已处理
- 转派或升级

## 3. 关键交互流程

### 3.1 新增应用流程

1. 从应用台账列表点击“新增应用”
2. 填写基础信息和责任矩阵
3. 选择所属项目
4. 填写依赖摘要或进入后续详情维护依赖对象
5. 保存成功后跳转详情页

### 3.2 维护依赖关系流程

1. 进入应用详情页
2. 打开“依赖关系”页签
3. 分别添加项目、IT 资源、数据库/API、网络资源
4. 保存后生成依赖摘要与状态联动基础

### 3.3 处理应用异常流程

1. 系统接收或汇总底层异常
2. 生成应用通知并推送责任人
3. 责任人在通知中心或应用详情查看异常
4. 标记处理中、补充备注、确认恢复
5. 系统保留处理留痕

### 3.4 查看统计分析流程

1. 用户进入统计分析页
2. 选择建设来源、单位、状态等筛选条件
3. 查看分布图与列表明细
4. 导出结果或进入详情追踪异常应用

## 4. 状态设计

### 4.1 应用主状态

- `BUILDING`：建设中
- `RUNNING`：运行中
- `ALERT`：告警中
- `DISABLED`：停用

### 4.2 通知处理状态

- 待处理
- 处理中
- 已处理
- 已升级

### 4.3 依赖对象展示状态

- 正常
- 告警
- 不可达
- 未接入监控

## 5. 异常流说明

- 应用编码重复：阻止保存并提示重复编码
- 所属项目失效：阻止继续绑定失效项目，并提示重新选择
- 未维护关键责任人：阻止保存并提示缺失责任角色
- 依赖对象不存在：在依赖关系页签中提示对象已失效，要求重新绑定
- 监控数据缺失：运行概览展示“暂无监控数据”，但不阻塞台账查看
- 用户无权限：页面显示只读态或无权提示，并隐藏对应操作按钮
